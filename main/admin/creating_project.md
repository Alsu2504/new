# Как создать проект

## Создать проект

Чтобы создать проект, в личном кабинете нажмите кнопку Создать проект. &#x20;

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2026-06-05 в 13.50.32.png" alt=""><figcaption></figcaption></figure></div>

Далее будет предложено пройти короткий опрос для настроек проекта:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2026-06-05 в 13.51.22.png" alt=""><figcaption></figcaption></figure></div>

На основе данного опроса будут настроены пункты меню и онбординга для быстрого старта.

Далее пропишите название проекта, выберите часовой пояс и нажмите на кнопку "Создать проект":

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2026-06-05 в 13.54.04.png" alt=""><figcaption></figcaption></figure></div>

{% hint style="success" %}
Если у вас есть промокод его можно ввести только при создании проекта.&#x20;
{% endhint %}

После создания проекта Вы увидите план-обучение, как быстро запустить проект в Salebot по вашим выбранным настройкам:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2026-06-05 в 13.57.06.png" alt=""><figcaption></figcaption></figure></div>

Если вы являетесь уверенным пользователем Salebot, то можно кликнуть по кнопке "Я хочу настроить проект сам", чтобы не отображать онбординг-обучение быстрого старта:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2026-06-05 в 14.01.33.png" alt=""><figcaption></figcaption></figure></div>

{% hint style="info" %}
Настройки таймеров и рассылки будут планироваться по часовому поясу указанному в настройках.&#x20;
{% endhint %}

* [x] Поздравляем! Вы создали свой первый проект.

{% hint style="warning" %}
**Обратите внимание!**

Оплата идёт **за проект, а не за аккаунт.**&#x20;

В одном аккаунте у вас может быть любое количество проектов, но действующими будут только оплаченные!
{% endhint %}

### Как настроить меню?

В проекте слева вы видите основные разделы проекта, которые отображаются на основе настроек, выбранных [на этапе создания проекта](creating_project.md):

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2026-06-05 в 14.09.23.png" alt=""><figcaption></figcaption></figure></div>

Чтобы добавить или убрать пункты меню, ниже слева найдите кнопку "Настроить пункты меню" и нажмите на нее:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2026-06-05 в 14.07.35.png" alt=""><figcaption></figcaption></figure></div>

Откроется форма с пунктами, которые можно добавить или убрать:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2026-06-05 в 14.16.28.png" alt=""><figcaption></figcaption></figure></div>

Выберите необходимые пункты и нажмите "Сохранить".

### Как узнать номер проекта?

Номер вашего проекта может понадобиться , чтобы сотрудники технической поддержки могли увидеть наглядно, какая ошибка есть в схеме конструктора, или подсказать, на что вам стоит обратить внимание при разработке веб-инструмента.

Номер проекта можно найти:

1. В разделе "Проекты" в личном кабинете:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-02-06 в 11.43.01.png" alt=""><figcaption></figcaption></figure></div>

В разделе "Проекты" в ЛК вы увидите созданные вами проекты. Номер проекта находится на информационной плашке проекта:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-02-06 в 11.44.31.png" alt=""><figcaption></figcaption></figure></div>

2. Номер проекта в адресной строке:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-02-06 в 11.46.20.png" alt=""><figcaption></figcaption></figure></div>

Чтобы каждый раз, когда вам понадобится искомый номер проекта, не переходить в раздел "Проекты" в личном кабинете, достаточно просто кликнуть по адресной строке вашего браузера.&#x20;

В адресной строке вы можете найти айди вашего проекта:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-02-06 в 11.48.07.png" alt=""><figcaption></figcaption></figure></div>

Номер проекта в адресной строке можно просмотреть в любом разделе Сейлбота внутри проекта:

<div><figure><img src="../../.gitbook/assets/Снимок экрана 2025-02-06 в 11.49.30.png" alt=""><figcaption><p>Пример: адресная строка в разделе "Задачи"</p></figcaption></figure> <figure><img src="../../.gitbook/assets/Снимок экрана 2025-02-06 в 11.50.46.png" alt=""><figcaption><p>Пример: адресная строка в разделе "Клиенты"</p></figcaption></figure></div>
