# Контент страницы сайта

Любой сайт (многостраничный или лендинг) состоит из страниц(ы), которая содержит в себе определенный контент.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Запись экрана 2025-06-16 в 10.41.36 (online-video-cutter.com).gif" alt=""><figcaption></figcaption></figure></div>

Контент страницы состоит из секций (блоков), которые помогают наполнить ваш сайт различными интересными элементами интерфейса: тексты, изображения, кнопки, форма сбора заявок, виджеты, карта вашего положения и многое другое.&#x20;

{% hint style="success" %}
В конструкторе Сейлбот существуют различные типы блоков, где каждый блок (секция) создается на сайте с собственными настройками:

* [Универсальный блок (](sekcii-bloki/universalnyi-blok-zero-block/)зеро блок): блок без предустановленных настроек - можно разместить любой контент и самые разные элементы сайта по своему уникальному дизайну, при этом элементы могут быть расположены в любом количестве и в любом порядке, в отличие от иных блоков (секций) сайта. Зеро блок может выступать в качестве любой секции сайта, будь то меню, футер, форма сбора заявок, галереи и т.п.
* [Обложка: ](sekcii-bloki/sekciya-oblozhka.md)блок для добавления изображение и видео с заголовком.\
  Традиционно блок обложки используется  в качестве визитной карточки сайта, с которым первоочередно знакомятся посетители вашего сайта.
* Заголовок и текст: текстовый блок с заголовком.&#x20;
* [Меню](sekcii-bloki/sekciya-menyu.md): для создания футера с кнопками в начале сайта (можно сделать плавающее меню).&#x20;
* [Онлайн-запись](../../onlain-zapis/vidzhet-onlain-zapisi-dlya-saita/): для добавления формы онлайн-записи для клиентов (для уже настроенных филиалов) с автоматическим подключением к SalebotCRM.
* [Этапы](sekcii-bloki/inye-sekcii-saita.md#sekciya-etapy) для создания уникального дизайна сайта с хронологией событий.
* [Аккордеон:](sekcii-bloki/sekciya-akkordeon.md) для добавления информации или данных на сайт в вопросно-ответной форме.
* [HTML:](sekcii-bloki/inye-sekcii-saita.md#sekciya-html) блок позволяет добавить на страницу HTML и CSS коды
* [Форма: ](sekcii-bloki/sekciya-forma/)блок для добавления кнопок мессенджеров, вопросов, политики конфиденциальности.&#x20;
* [Квиз формы](sekcii-bloki/sekciya-kviz.md): блок для создания постраничных опросов (квизов) на сайте.&#x20;
* [Медиа:](sekcii-bloki/inye-sekcii-saita.md#sekciya-media) блок для добавления изображений или видео
* [Кнопки: ](sekcii-bloki/sekciya-knopki.md)блок для добавления кнопок с функцией ссылки, телефона, email, оплаты
* [Цитата](sekcii-bloki/inye-sekcii-saita.md#sekciya-citata): блок для добавления цитаты
* [Список](sekcii-bloki/inye-sekcii-saita.md#sekciya-spisok): блок с возможностью вставить нумерованные и маркированные списки
* [Выпадающий текст:](sekcii-bloki/inye-sekcii-saita.md#sekciya-vypadayushii-tekst) блок для добавления выпадающего текста
* [Галерея:](sekcii-bloki/inye-sekcii-saita.md#sekciya-galereya) блок для добавления галереи изображений.&#x20;
* [Pop-up:](sekcii-bloki/sekciya-pop-up.md) для добавления всплывающего окна на сайте
* [Настраиваемые карточки](sekcii-bloki/sekciya-nastraivaemye-kartochki.md): поможет добавить отзывы клиентов или информацию о товаре.
* [Карточки курсов](sekcii-bloki/sekcii-dlya-onlain-kursa.md#kak-dobavit-blok-kartochki-kursa) понадобится для продажи курсы прямо с сайта: вам не нужно самим собирать и настраивать карточку, данные подтягиваются из раздела "Онлайн-курсы".&#x20;
* [Список уроков](sekcii-bloki/sekcii-dlya-onlain-kursa.md#kak-dobavit-blok-spiski-urokov): достаточно выбрать курс в настройках блока и все уроки этого курса отобразятся на сайте
* [Таймер](sekcii-bloki/inye-sekcii-saita.md#sekciya-taimer): блок для добавления таймера обратного отсчета.
* [Таблицы:](sekcii-bloki/sekciya-tablicy.md) для добавления структурированной информации на сайт в виде таблиц.
* [Cookie](sekcii-bloki/sekciya-uvedomlenie-cookie.md): блок для прозрачной работы сайта для пользователей, который показывает, что вы честно информируете, какие данные сохраняются с сайта и как они помогают улучшить взаимодействие вас и ваших пользователей.
{% endhint %}

Чтобы наполнить страницу сайта контентом из секций, нужно перейти в режим редактирования страницы, что можно сделать следующим образом:

1. Многостраничный сайт

Для перехода в режим редактирования страницы сайта нажмите на плашку нужного сайта для перехода в его настройки:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 13.35.59.png" alt=""><figcaption></figcaption></figure></div>

{% hint style="info" %}
В случае, если вы только создали новый многостраничный сайт, Вам никуда переходить не нужно, поскольку вы уже будете находиться во вкладке с настройками и страницами сайта.
{% endhint %}

Далее нажмите на нужную страницу сайта:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 13.37.47.png" alt=""><figcaption></figcaption></figure></div>

Тогда вам откроется форма редактирования страницы, где можно добавить/удалить блоки (секции) сайта, поменять шрифты, фон страницы и пр.:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 13.38.32.png" alt=""><figcaption></figcaption></figure></div>

2. Лендинг

При создании одностраничного сайта вы сразу перейдете в режим редактирования страницы сайта:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 13.55.23.png" alt=""><figcaption></figcaption></figure></div>

В случае, если необходимо отредактировать уже существующую страницу, то перейдите в раздел "Сайты", найдите необходимый лендинг (он будет помечен как "Лендинг" на плашке) и нажмите на него:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 13.56.46 (1).png" alt="" width="563"><figcaption></figcaption></figure></div>

Далее вы перейдете в настройки и основную информацию о сайте. Здесь необходимо кликнуть по кнопке "Редактировать":

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 13.58.18.png" alt=""><figcaption></figcaption></figure></div>

Аналогично вы сможете перейти в режим редактирования страницы с формой, опросом и квизом. При этом в общем разделе "Сайты" вы сможете увидеть, к какому типу относится тот или иной сайт:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 14.16.06.png" alt=""><figcaption></figcaption></figure></div>

{% hint style="success" %}
**Горячие клавиши для входа в полноэкранный режим редактирования:**

`Ctrl+shift+X` или `Cmd+shit+X`
{% endhint %}

## Как добавить новую секцию (блок)

Чтобы добавить новый блок на странице, наведите на рабочую область страницы сайта и найдите кнопку "+":

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 13.41.28.png" alt=""><figcaption></figcaption></figure></div>

При клике на кнопку "+" откроется правое меню с выбором блока, который нужно добавить на страницу:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 13.42.58.png" alt=""><figcaption></figcaption></figure></div>

{% hint style="info" %}
Подробно о настройке каждого блока (секции) сайта рассказали в одноименном разделе "[Секции (блоки](sekcii-bloki/))".
{% endhint %}

## Как удалить секцию (блок)

Чтобы удалить ненужный блок, наведите на него и найдите соответствующую кнопку:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 13.49.03.png" alt=""><figcaption></figcaption></figure></div>

Нажмите на кнопку "удалить", тогда секция будет безвозвратно удалена.&#x20;

{% hint style="warning" %}
Важно!&#x20;

Удаленную секцию (блок) нельзя восстановить!
{% endhint %}

## Как скрыть секцию (блок)

В случае, если вы пока не хотите удалять секцию (блок) или нужно скрыть ее на какое-то время, то наведите на блок и найдите кнопку "выкл":

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 13.51.09.png" alt=""><figcaption></figcaption></figure></div>

При клике на кнопку секция будет скрыта:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 13.52.16.png" alt=""><figcaption></figcaption></figure></div>

Чтобы вернуть секцию, нажмите эту же кнопку.

## Как скопировать секцию

Чтобы скопировать секцию (например, для настройки разного [диапазона видимости на устройствах](kontent-stranicy-saita.md#diapazon-vidimosti)), вам понадобится скопировать секцию, добавив ее на эту же страницу.&#x20;

Чтобы скопировать секцию, наведите на нее мышью и найдите кнопку для копирования:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 14.16.47.png" alt=""><figcaption></figcaption></figure></div>

При клике на кнопку "Скопировать" секция будет добавлена ниже.

## Как редактировать секцию (блок)

Чтобы перейти в режим редактирования секции, наведите на нее мышью и кликните на кнопку "Редактировать":

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 14.20.22.png" alt=""><figcaption></figcaption></figure></div>

В зависимости от типа блока:

1. Вам откроется меню для редактирования контента блока справа (для всех блоков, кроме универсального):

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 14.21.50.png" alt=""><figcaption></figcaption></figure></div>

2. Вам откроется страница с редактированием универсального блока (зеро блока) в отдельном окне:

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 14.23.28.png" alt=""><figcaption></figcaption></figure></div>

{% hint style="success" %}
Чтобы знать, как отредактировать любой блок и для чего нужны какие-либо функции и настройки, ознакомьтесь с разделом "[Секции (блоки)](sekcii-bloki/)".
{% endhint %}

## Настройки секции (блока)

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 15.08.45.png" alt=""><figcaption></figcaption></figure></div>

Если сам контент секции редактируется при клике на кнопку "Редактировать", то остальные настройки (шрифтов, положения элементов блока, отображение секции и т.п.) осуществляются при клике на кнопку "Настройки" (для всех блоков, кроме универсального):

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 15.10.04.png" alt=""><figcaption></figcaption></figure></div>

Настройки шрифтов, фона и отображения являются универсальными настройками — это значит что такие настройки схожи между собой в разных секциях (блоках).

{% hint style="success" %}
Подробнее об универсальных настройках шрифтов, фона и контента рассказали в одноименной статье в разделе "[Секции (блоки)](sekcii-bloki/)".
{% endhint %}

#### Настройка отображения

С помощью данного меню можно настроить отображение данного блока страницы и подписной ВК. А также сделать оптимизацию под любой тип устройств: настроить видимость блока в зависимости от размера экрана устройства.

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-03-19 в 14.13.15.png" alt="" width="563"><figcaption></figcaption></figure></div>

{% hint style="warning" %}
У подписной страницы <mark style="color:orange;">**ВКонтакте**</mark> фиксированные размеры, поэтому параметр <mark style="color:orange;">**Ширина секции и Ширина контента работать НЕ БУДУТ**</mark> (значения не применятся).
{% endhint %}

#### Диапазон видимости

В настройках отображения секции (блока) есть возможность указать для устройств при каком размере экрана будет отображаться текущая секция(блок).

<div data-with-frame="true"><figure><img src="../../.gitbook/assets/Снимок экрана 2025-06-16 в 15.15.39.png" alt=""><figcaption></figcaption></figure></div>

Таким образом можно для мобильных устройств сделать более адаптированную версию контента данной секции (блока):

<div><figure><img src="../../.gitbook/assets/Снимок экрана 2025-03-19 в 14.14.57.png" alt=""><figcaption></figcaption></figure> <figure><img src="../../.gitbook/assets/Снимок экрана 2025-03-19 в 14.15.14.png" alt=""><figcaption></figcaption></figure></div>

Таким образом, можно создать одну секцию, скопировать ее несколько раз и адаптировать уже готовые секции под различные размеры устройств.

Пример: одинаковые секции созданы на странице для разных устройств. При этом, первая секция отображается только для устройств, размер экрана которых не превышает 960 px.

## Как встроить кнопки к Вам на сайт

{% endhint %}
